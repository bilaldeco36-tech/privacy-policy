# Privacy Policy — Insights Auto Flow

Insights Auto Flow does not collect, store, transmit, or sell any personal user data. All prompts, settings, and generated files remain locally on the user's own device and browser. The extension interacts only with Google Flow (labs.google) using the user's existing logged-in session to automate generation tasks the user initiates. No data is sent to the developer or any third party.
